Platform Features

Everything you need to manage your notes securely.

Secure Accounts

Complete user authentication system with hashed passwords and secure sessions.

Markdown Supported

Write your notes in raw Markdown, and our platform renders them beautifully with code blocks and tables.

Public & Private

Keep sensible thoughts private in your dashboard, or mark notes as public to share with the community!

Explore & Search

Paginated exploration feed allows you to keyword-search through all public thoughts generated by users.

REST API

Interact with the database programmatically via a sleek RESTful JSON API using /api endpoints.

Python + Flask

Built dynamically using standard Python libraries, SQLAlchemy ORM, and Jinja2 Templating.

Deploy It Now
